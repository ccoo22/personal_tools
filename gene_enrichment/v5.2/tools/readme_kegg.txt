标题	说明
ID	通路编号
Description	通路描述
GeneRatio	富集基因在该通路上的比例
BgRatio	背景基因在该通路上的比例
pvalue	P值
p.adjust	校正P值（BH方法校正后的P值）
qvalue	Q值（Q方法校正后的P值）
geneID	通路上所有富集基因的kegg id，用“/”分割。
Count	通路上所有富集基因的数量
geneSymbol	通路上所有富集基因的geneSymbol，用“/”分割。geneSymbol与geneID从左到右按顺序一一对应。
keggLink	通路图kegg链接地址（标记颜色）
level_a	kegg 通路分级A注释(层级关系参考：https://www.kegg.jp/kegg/pathway.html)
level_b	kegg 通路分级B注释
